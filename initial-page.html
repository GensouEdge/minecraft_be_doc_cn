注意

文档原文就存在些许问题，以下为一些常见问题：

  • minecraft:equippable 中的 on_equip 和 on_unequip 的类型应该是 JSON Object，但在文档原文中显示的类型却是 String。
  • 幽默 Mojang 在正文中会使用 <>，例如： definitions/entity/<entity_name>.json，但浏览器会将 <> 和其中的内容当作 HTML 标签处理，导致 <entity_name> 被当作标签屏蔽。此类情况可以使用转义符 &lt; 和 &gt; 来表示 < 和 >。
